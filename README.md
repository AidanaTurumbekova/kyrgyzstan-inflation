# Kyrgyzstan Inflation & Economic Growth Analysis Dashboard


## Overview

This project is an interactive analytical dashboard developed to analyze inflation, GDP growth, purchasing power, and future economic scenarios in Kyrgyzstan using historical economic data.

The dashboard combines data analysis, visualization, and forecasting tools to help users better understand long-term economic trends and the impact of inflation on the economy and daily life.

---

## Features

* Historical inflation analysis
* GDP growth analysis
* Inflation vs GDP comparison
* Purchasing power analysis
* Interactive visualizations
* Scenario-based forecasting
* Inflation prediction models
* Dynamic dashboard interface

---

## Technologies Used

* Python
* Pandas
* NumPy
* Plotly
* Streamlit
* Scikit-learn
* World Bank Data API / datasets

---

## Dataset

The project uses publicly available economic indicators from the World Bank related to:

* Inflation rate
* GDP growth
* Consumer price index (CPI)
* Real GDP per capita growth

The analyzed data mainly covers the period from 1996 to 2024.

---

## Main Goal

The main goal of this project is to create an accessible and interactive system for analyzing how inflation affects economic growth and purchasing power in Kyrgyzstan.

---

## Dashboard Sections

### Historical Analysis

Visualizes inflation and GDP trends over time.

### Purchasing Power Analysis

Shows how inflation changes the real value of money across different years.

### Forecasting

Implements regression-based prediction models and future economic scenarios.

### Scenario Simulator

Allows users to explore optimistic, base case, and pessimistic inflation scenarios.
